# Пример использования NativeBPM .NET SDK

Этот пример демонстрирует интеграцию сгенерированного клиента .NET Client SDK в консольное приложение .NET с использованием Microsoft Dependency Injection.

## Требования

- Установленный в системе **.NET 8.0 SDK**.
- Запущенный инстанс **NativeBPM Server** (обычно по адресу `http://localhost:8080`).

## Конфигурация проекта

Проект ссылается на локальную библиотеку SDK-клиента с помощью элемента `<ProjectReference>` в файле `.csproj`.

## Запуск примера

Убедитесь, что сервер NativeBPM запущен, и выполните следующую команду в терминале:

```bash
dotnet run
```

Это запустит программу, которая выполнит следующие шаги:
1. Регистрирует `IDefaultApi` с помощью метода расширения Microsoft `IServiceCollection` под названием `AddApi`.
2. Настраивает базовый адрес на `http://localhost:8080` и добавляет токен авторизации Bearer.
3. Автоматически создает временный XML-файл BPMN.
4. Развертывает схему процесса на сервере с помощью `DeployDefinitionAsync`.
5. Запускает инстанс процесса для развернутой схемы с помощью `StartInstanceAsync`.
6. Запрашивает и выводит список всех развернутых определений процессов с помощью `ListDefinitionsAsync`.
7. Запрашивает и выводит список всех активных инстансов с помощью `ListInstancesAsync`.
