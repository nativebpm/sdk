# Приводящий пример Java SDK NativeBPM

В этом каталоге находится готовое консольное приложение на Java, демонстрирующее интеграцию с REST API движка NativeBPM с использованием сгенерированного Java SDK клиента.

## Требования
- **Java JDK 17** или выше
- **Maven 3.6** или выше
- Запущенный инстанс **NativeBPM Server** на `http://localhost:8080` (с токеном `test-bearer-token`)

## Сборка и запуск

1. **Установите SDK в локальный кэш Maven (`~/.m2`)**:
   Перед компиляцией примера соберите и установите сам SDK клиент:
   ```bash
   cd ../
   mvn clean install
   ```

2. **Запустите пример**:
   Вернитесь в эту папку и выполните команду Maven:
   ```bash
   cd examples
   mvn compile exec:java
   ```

Скрипт автоматически создаст временный файл BPMN, развернет его на сервере, запустит инстанс процесса с тестовыми переменными, а также выведет списки развернутых определений и активных инстансов с сервера.
